Ubuntu 开发环境搭建以及常用软件安装

Posted on 2016-04-08 Edited on 2019-12-06 In Ubuntu Views: Valine:

最近开发工具要从Mac切换到Ubuntu，开始学习Chromium内核相关知识，本文主要纪录Ubuntu（12.04LTS）安装完成之后，安装常用软件的过程，将 Android 开发环境搭建成功。

zsh 与oh－my－zsh

Mac 上一直使用，为保持一致的体验，首先选择安装zsh。zsh与oh-my-zsh安装方法参考

安装完成之后记得重启电脑。

JDK

Android 开发，怎么能没有JDK。JDK 安装方法参考

这里有个小坑，使用 apt-get 安装 jdk 之后，编译 chromium 内核时，提示无法找到 java相关的命令，但是在终端里自行 java,javac 等这些命令，都是正常的。后来通过手动安装jdk，在 .zshrc 中配置下环境变量，再编译就OK了。通过apt－get 安装配置下环境变量理论上也是可以的，不过没有实验。

Android Studio

在 Ubuntu 下，Android Studio 安装也是傻瓜式的，解压之后，找到 studio.sh ,确保有执行权限，执行打开即可。打开之后，会提示下载SDK，各种支持库等等。
Android Studio 安装之后，有一些常用的插件
